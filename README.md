# E-commerce-Analysis-Dashboard
This project presents a comprehensive sales and profitability analysis for an e-commerce company specializing in bicycles and related accessories. Using tools like Excel, Tableau, and SQL, this project explores how product categories, subcategories, distance, and time affect sales performance.

 📌 Objective

The objective of this project is to:
- Understand total sales, profit, tax, and freight costs across multiple years
- Analyze sales by product categories and subcategories
- Discover the impact of delivery distance on customer behavior
- Track profitability by product and over time
- Build an interactive, data-driven dashboard for decision-makers

---

## 🧰 Tools & Technologies Used

- **Excel**: Data cleaning, transformation, and preparation  
- **SQL (MySQL)**: For querying, aggregating, and filtering transactional data  
- **Tableau**: For building interactive visualizations and dashboards  

---

## 📦 Dataset Overview

The dataset used includes transactional and order data for bikes and accessories, with key fields such as:

- Order Date  
- Product Category & Subcategory  
- Sales, Profit, Tax Amount, Freight Cost  
- Order Quantity  
- Delivery Distance  
- Customer Location  


## 📈 Key Insights

- **Mountain Bikes** and **Road Bikes** generate the highest sales and profit.
- The majority of revenue comes from customers located within 0–1 mile radius.
- Profitability significantly increased in 2013, peaking in Q4.
- Freight and tax costs remain relatively stable despite growth in sales volume.

---

## 📍 Conclusion

This project demonstrates how data analysis and visualization can uncover meaningful insights from e-commerce sales data. The dashboard provides a centralized view for tracking key KPIs and improving business strategy.
